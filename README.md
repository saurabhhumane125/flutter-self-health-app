# Flutter Self Health App

A Flutter-based Self Health Recording application that allows users to track daily mood, view history, and manage entries with a clean animated UI.

## Features
- Daily mood tracking
- Animated mood selection UI
- Health summary with insights
- Mood history view
- Edit and delete mood entries
- Clean, health-focused design

## Tech Stack
- Flutter
- Dart
- Material UI

## Platform
- Flutter Web (demonstration)
- Android-ready architecture
